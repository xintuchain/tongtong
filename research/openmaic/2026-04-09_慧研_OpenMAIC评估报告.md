## 一、项目概览

| 指标 | 数据 |
|------|------|
| 项目名称 | OpenMAIC (Open Multi-Agent Interactive Classroom) |
| 开发机构 | 清华大学 (THU-MAIC) |
| GitHub | https://github.com/THU-MAIC/OpenMAIC |
| 许可证 | AGPL-3.0 |
| 最新版本 | v0.1.0 (2026-03-26) |
| 技术栈 | Next.js 16, React 19, TypeScript 5, LangGraph 1.1 |

---

## 二、核心功能

### 🚀 主要特性

| 功能 | 说明 |
|------|------|
| 一键课程生成 | 输入主题或上传材料，AI 构建完整课程 |
| 多智能体教室 | AI 教师和同学实时授课、讨论、互动 |
| 丰富场景类型 | 幻灯片、测验、交互式模拟、PBL项目式学习 |
| 白板 & TTS | AI 画图、写公式、口头讲解 |
| 多格式导出 | 下载 .pptx 幻灯片或交互式 .html 页面 |
| **OpenClaw 集成** | 从飞书、Slack、Telegram 等消息应用直接生成教室 |

### 🤖 支持的 LLM 提供商

- OpenAI (GPT)
- Anthropic (Claude)
- Google Gemini
- DeepSeek
- MiniMax
- Grok (xAI)
- 任何 OpenAI 兼容 API

## 四、局限性分析

| 局限 | 说明 | 对我们的影响 |
|------|------|-------------|
| 专注教学场景 | 定位是"互动教室"，非通用知识管理 | 需要二次开发 |
| 依赖 LLM API | 需要配置外部 API 密钥 | 成本考虑 |
| 自托管需求 | 完整功能需要本地部署 | 技术门槛 |
| AGPL-3.0 许可 | 商业使用需注意许可条款 | 法律合规审查 |

## 五、安装与使用

### OpenClaw Skill 安装（推荐）

```bash
clawhub install openmaic
```

### 手动部署

```bash
git clone https://github.com/THU-MAIC/OpenMAIC.git
cd OpenMAIC
pnpm install
cp .env.example .env.local
# 配置 LLM API 密钥
pnpm dev
```

- **在线体验**: https://open.maic.chat/
