# Contributing to Vision Sandbox

We welcome contributions! Since this is an AI-native skill, we appreciate both human and agent-led improvements.

## How to Contribute

1. **Fork the Repository:** Visit [github.com/johanesalxd/vision-sandbox](https://github.com/johanesalxd/vision-sandbox).
2. **Local Development:**
   - Clone your fork.
   - Use `uv sync` to set up your environment.
   - Follow the guidelines in `AGENTS.md` for technical standards and code style.
3. **Submit a PR:** Provide a clear description of your changes and why they are useful.

## Areas for Improvement
- Adding more visual reasoning patterns to `SKILL.md`.
- Improving error handling for network/API failures.
- Implementing a proper test suite with `pytest`.

Thank you for helping make Agentic Vision more accessible!
