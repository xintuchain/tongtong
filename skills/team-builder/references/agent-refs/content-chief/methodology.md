# 内容方法论

## 内容漏斗匹配（Content-Funnel Fit）

每篇内容明确服务哪个用户旅程阶段：

| 阶段 | 用户意图 | 内容类型 | 示例 |
|------|---------|---------|------|
| **TOFU** 认知 | "有这类工具吗？" | 科普、趋势、问题解答 | "2026年影视数据库对比" |
| **MOFU** 考虑 | "哪个适合我？" | 对比评测、教程、案例 | "Richlyva vs IMDb 角色图谱对比" |
| **BOFU** 决策 | "我要用这个" | 上手指南、API文档 | "5分钟搭建影视宇宙" |

**比例约 5:3:2（TOFU:MOFU:BOFU）**

## 内容日历与排期

### 发布节奏（冷启动期）
| 类型 | 频率 | 负责 | 说明 |
|------|------|------|------|
| 长文（>1500词） | 2篇/周 | 内容主编主导 | SEO/GEO核心资产 |
| 短文/列表（500-1000词） | 2篇/周 | 可委托或模板化 | 长尾关键词覆盖 |
| 社区帖（Reddit/HN/Quora） | 3条/周 | 配合growth-lead | 引流+品牌曝光 |
| FAQ/更新日志 | 按需 | 配合product-lead | 站内SEO+用户信任 |

### 选题流程（4步）
1. **关键词池** → 从 SEO/GEO 关键词研究中选取本周目标词
2. **意图匹配** → 判断搜索意图（信息型/导航型/交易型）→ 匹配漏斗阶段
3. **竞品内容差距** → intel-analyst 竞品数据中找对手没覆盖的角度
4. **排期** → 写入 `shared/knowledge/content-plan-{YYYY}-W{nn}.md`

### 内容日历模板
```markdown
## 内容计划 {YYYY}-W{nn}
| 日期 | 标题 | 漏斗 | 目标关键词 | 渠道 | 状态 |
|------|------|------|-----------|------|------|
| 周一 | ... | TOFU | ... | Blog | 草稿/发布/推迟 |
```

## SEO/GEO 关键词研究流程

### 第1步：种子词收集
- 产品核心功能词（character map, movie universe, film database）
- 用户痛点词（从 data-analyst 反馈/搜索query）
- 竞品品牌词+替代词（IMDb alternative, letterboxd vs）
- AI搜索常见问法（best tool for..., how to find...）

### 第2步：扩展与验证
| 工具/方法 | 用途 | 成本 |
|-----------|------|------|
| Google Suggest / People Also Ask | 长尾扩展 | 免费 |
| Bing Webmaster | 实际搜索数据 | 免费 |
| 竞品页面H1/H2/Title扫描 | 发现覆盖差距 | 免费 |
| Reddit/Quora热门问题 | 真实用户措辞 | 免费 |
| Google Trends | 趋势验证 | 免费 |

### 第3步：关键词评分
| 因子 | 说明 | 权重 |
|------|------|:----:|
| 搜索量预估 | 高/中/低（无精确数据时用趋势判断） | 30% |
| 竞争难度 | SERP前10是大站还是小站 | 25% |
| 意图匹配 | 和产品功能的相关度 | 25% |
| GEO潜力 | AI搜索是否会引用此类内容 | 20% |

### 第4步：关键词档案
```markdown
## 关键词: {keyword}
- 搜索量预估: 高/中/低
- 竞争难度: 高/中/低
- 意图: 信息型/导航型/交易型
- 漏斗阶段: TOFU/MOFU/BOFU
- SERP现状: {前3结果概述}
- GEO机会: {AI搜索是否已有引用}
- 优先级: P1/P2/P3
- 计划内容: {标题草案}
```

## 内容效果评估框架

### 发布后追踪（分阶段）
| 时间点 | 检查项 | 合格线 |
|--------|--------|--------|
| 发布即时 | Google索引（site:命令） | 48h内收录 |
| 7天 | Search Console曝光量 | >0（有曝光） |
| 14天 | 排名位置 | 目标词进前50 |
| 30天 | 点击量、平均排名 | 目标词进前20 |
| 60天 | 自然流量占比 | 该页有稳定流量 |

### 内容健康度分级
| 等级 | 标准 | 行动 |
|------|------|------|
| 🟢 健康 | 排名稳定/上升，有流量 | 保持，考虑扩展内链 |
| 🟡 待优化 | 排名11-30，有曝光无点击 | 优化标题/摘要/结构 |
| 🔴 失败 | 60天无排名无流量 | 分析原因：选题错？内容薄？竞争强？ |
| ⚫ 过时 | >6个月未更新且流量下降 | 更新或合并 |

### 内容ROI评估
```
内容价值 = (自然流量 × 预估CPC) / 创作时间(小时)
```
无精确数据时用定性判断：这篇内容带来了注册/咨询/品牌曝光吗？

## 内容复用策略（1→N）

一篇核心长文可拆解为：

| 衍生形式 | 渠道 | 改造量 |
|---------|------|--------|
| 精华摘要（300字） | Reddit/HN帖子 | 低 |
| FAQ拆条 | Quora回答 | 低 |
| 数据可视化 | Twitter/社媒图 | 中 |
| 对比表格 | 独立落地页 | 中 |
| 视频脚本大纲 | YouTube（未来） | 高 |
| Newsletter片段 | 邮件（未来） | 低 |

**规则：核心长文先发博客 → 等收录后 → 再拆分发各渠道（避免重复内容惩罚）**

- [ ] 内链 ≥2 个
- [ ] CTA 明确

- [ ] 标注：目标关键词、漏斗阶段、目标AARRR环节

- [ ] URL slug 简短含关键词（英文短横线分隔）
