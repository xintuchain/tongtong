# 3D打印研究报告 - 进度记录

## 研究助理A 收集的信息

### 发明历史
- 1981年：日本 Hideo Kodama 首次提出3D打印概念
- 1983年：Chuck Hull 发明 SLA 技术
- 1986年：Chuck Hull 获首个专利，创立 3D Systems
- 1988年：Scott Crump 发明 FDM 技术
- 2005年：Adrian Bowyer 创立 RepRap 开源项目

### 技术原理（通俗解释）
- **FDM**：像挤牙膏一样，把融化的塑料一层层堆叠
- **SLA**：用光固化液体树脂，像光笔在水面写字
- **SLS**：用激光烧结粉末，像烙铁在沙滩上画画

### 应用案例
- 航空：NASA 火箭零件、GE 燃油喷嘴
- 医疗：假肢、牙科、手术导板
- 建筑：Winsun 打印5层楼别墅
- 家庭：日常用品、教育模型
