## 🔄 工作流程

### 当前实际工作流

```
Fei 指令（微信/控制台）
    ↓
桐桐 接收 + 分解任务
    ↓
Discord 频道分配任务
    ↓
各成员执行 → 汇报进度
    ↓
桐桐 汇总 + 交付
    ↓
Obsidian 存档
```

### 目标工作流（闭环）

```
外部用户 飞书提交任务
    ↓
桐桐/机器人 接收
    ↓
任务分解 + 分配
    ↓
成员执行 + 协作
    ↓
Obsidian 存档
    ↓
飞书 通知交付
```
